# Mountain Model Generator

A simple web-based tool to create 3D mountain range models with real-time parameter controls and STL export for 3D printing.

The controls include an option to keep the back side of the model flat so it can sit on a surface when printed.

## Development

This project uses only static files. To run a local server:

```bash
npm run start
```

Then open `http://localhost:8080` in your browser.
